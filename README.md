**Manzil – Smart Transportation Recommendation System for Pakistan**

Manzil is a Python-based transportation recommendation and fare estimation system designed to help users choose the most suitable mode of travel between cities in Pakistan. The application supports over **50 major cities**, calculates distances using the **Haversine formula**, estimates travel costs for different transportation options, and provides intelligent recommendations based on user preferences.

The project features a user-friendly **Tkinter GUI**, secure **bcrypt authentication**, and a **MySQL database** for managing users, city data, and travel records. Manzil aims to simplify travel planning by offering accurate distance calculations, fare comparisons, and personalized transport suggestions in a single platform.

### Features

* User registration and login with bcrypt password hashing
* Support for 50+ cities across Pakistan
* Distance calculation using geographic coordinates
* Fare estimation for multiple transport modes
* Smart transport recommendations
* Search history storage and retrieval
* Interactive Tkinter graphical interface
* MySQL database integration

### Technologies Used

* Python
* Tkinter
* MySQL
* bcrypt
* Git & GitHub

This project was developed as a transportation management and recommendation solution to assist travelers in making informed and cost-effective travel decisions across Pakistan.
